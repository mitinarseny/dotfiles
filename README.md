## Install

> **NOTE:** make sure the following is installed:
> * [curl](https://curl.se/)
> * [git](https://git-scm.com/)
> * [make](https://www.gnu.org/software/make/)
> * [sudo](https://www.sudo.ws/)

```sh
$ git clone https://github.com/mitinarseny/dotfiles ~/.dotfiles
$ cd ~/.dotfiles
$ make -B
```
